# HR Playbooks — GCB

---

## 👥 PLAYBOOK 1: Contributor Onboarding

1. Join via GitHub or email
2. Assign role:
   - Content
   - Engineering
   - Research
3. Provide:
   - GCB Doctrine (Mirror → Water → Fire)
   - Contribution guidelines
4. Assign first task

---

## 👥 PLAYBOOK 2: DAO Participation

- Track contributions via commits
- Reward via recognition or tokens (non-investment framing)
- Log all activity in ATTESTATION_LOG.md

---

## 👥 PLAYBOOK 3: Role Scaling

Trigger: consistent high-quality output

→ Promote to:
- Lead contributor
- Reviewer
- Maintainer

---

## 👥 PLAYBOOK 4: Culture Enforcement

All contributors must align with:
- Truth (Mirror)
- Growth (Water)
- Execution (Fire)

Non-alignment → removal from system

---

## 👥 PLAYBOOK 5: Academic Layer (SydTek Scholars)

- Treat contributions as coursework
- Map:
  - GitHub commits → assignments
  - YouTube engagement → discussions
- Track learning outcomes
