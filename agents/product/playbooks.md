# Product Playbooks — GCB

---

## 📦 PLAYBOOK 1: Book Creation (GCB Series)

1. Define series:
   - Example: Footprints of the Messiah
2. Define framework:
   - Pattern → Prophecy → Fulfillment
3. Build:
   - Outline
   - Chapters
   - Supporting content (YouTube + Substack)

---

## 📦 PLAYBOOK 2: Course Creation (SydTek)

- Convert books into:
  - Modules
  - Lessons
  - Assignments

Include:
- Scripture prompts
- Reflection questions
- Discussion layer

---

## 📦 PLAYBOOK 3: Dataset + Knowledge Graph

- Convert content into:
  - JSON
  - Ontology
  - Entity relationships

Push to:
- GitHub
- Zenodo
- Dataverse

---

## 📦 PLAYBOOK 4: Certification Layer

- Bundle:
  - Course + assessment + participation
- Issue:
  - SydTek Scholars certification

---

## 📦 PLAYBOOK 5: Release System

Each product must include:
- GitHub release
- Substack announcement
- YouTube integration
- Email distribution
